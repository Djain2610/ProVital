
# 🧬 ProVital

**ProVital** is a modern, responsive health and wellness landing page built using **React**. It aims to provide users with essential health insights, services, and product information in a clean and engaging interface.

🔗 [Live Site](https://pro-vital-fmko.vercel.app/)  
📁 [Source Code](https://github.com/Djain2610/ProVital)

---

## 🚀 Tech Stack

- **Frontend**: React, JSX, CSS
- **Deployment**: Vercel
- **Tools**: Git, GitHub

## 🛠️ Setup Instructions

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Djain2610/ProVital.git
   cd ProVital
````

2. **Install Dependencies**

   ```bash
   npm install
   ```

3. **Run the App Locally**

   ```bash
   npm start
   ```

4. **Build for Production**

   ```bash
   npm run build
   ```
